## 1.ヘッダー

> H1~H6 ⇨#~#######

| 機能                   | Win/Linuxキー | MacOSキー    |
| :--------------------- | :------------ | :----------- |
| 数値に対応したH1〜H6   | Ctrl + (1〜6) | Cmd + (1〜6) |
| ヘッダーレベルを上げる | Ctrl + +      | Cmd + ^      |
| ヘッダーレベルを下げる | Ctrl + -      | Cmd + -      |
